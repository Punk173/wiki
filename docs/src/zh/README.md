### Q: 什么是People's Punk?
A: People’s Punk是由173位发起人共同发起的基于NFT社区的下一代社交网络，People's Punk的市值基于碎片化的Cryptopunk#173，是衡量这个社交网络的社交资本指标。

### Q:什么是$DDDD代币？
A:$DDDD是Cryptopunk#173碎片化后的代币，也是People’s Punk社交网络的代币。
